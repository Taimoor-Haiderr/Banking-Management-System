# Bank Management System

## Overview

Bank Management System is a console-based application developed in C++ using Object-Oriented Programming (OOP) concepts. The project is designed to perform basic banking operations in an efficient and user-friendly way.

## Features

* Create Savings and Current Accounts
* Deposit Money
* Withdraw Money
* Transfer Funds Between Accounts
* Balance Inquiry
* Transaction History
* Modify Account Details
* Delete Accounts
* Display All Accounts

## OOP Concepts Used

* Classes and Objects
* Inheritance
* Polymorphism
* Encapsulation
* Constructors and Destructors
* Vectors
* Pointers

## Technologies Used

* C++
* STL Vector Library
* Console-Based Interface

## Project Structure

* `Transaction Class` → Stores transaction details
* `Account Class` → Base class for account operations
* `SavingsAccount Class` → Derived class for savings accounts
* `CurrentAccount Class` → Derived class for current accounts
* `Bank Class` → Handles complete banking operations

## Purpose

This project was developed for learning and implementing OOP concepts in a real-world banking scenario. It demonstrates how classes and functions can be used to manage banking records efficiently.
